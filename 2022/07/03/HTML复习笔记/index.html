HTML文档笔记

超文本标记语言HyperText Markup Language–定义了网页内容的含义和结构

HTML 使用“标记”（markup）来注明文本、图片和其他内容，以便于在 Web 浏览器中显示

HTML 元素通过“标签”（tag）将文本从文档中引出<>

空元素void

不含任何内容的元素
比如img：这是因为图像元素不需要通过内容来产生效果，它的作用是向其所在的位置嵌入一个图像。

文档详解

  1. !DOCTYPE html–文档类型
    DOCTYPE–用来保证文档正常读取
  2. html–包含整个页面的内容–根元素
  3. head–元素的内容对用户不可见/搜索引擎的搜索关键字，页面描述，css样式表，字符编码声明
  4. meta charset=”utf-8”–指定文档使用的字符编码–包含人类已经知道的文本内容
  5. title–页面的标题/收藏网页的描述文字
  6. body–用户看到的内容

元数据–描述数据的数据

从对象到iframe

iframe–提供将整个web页面嵌入到另一个网页的方法，看起来就像那个web网页时另一个网页的一个img或者其他元素

作用

  1. 嵌入在线视频
  2. 广告横幅
  3. 在线地图提供商
  4. 评论系统

基本要素

  1. allowfullscreen：设置为全屏模式
  2. frameborder
  3. src–有利于seo
  4. width/height
  5. sandbox–提高安全性设置

安全隐患
攻击向量–将iframe作为攻击目标
–显示用户名和密码

最常见：
单击劫持：将隐藏的iframe嵌入到我的文档中，并使用它来捕获用户的交互

只在有必要的时候嵌入

  1. 安全问题
  2. 知识产权
  3. 引用–注明来源

使用https

  1. 减少远程内容在传输过程中被篡改的机会
  2. 防止嵌入式内容访问你的父文档中的呢日哦那个

sandbox属性
沙盒：允许包含在其中的代码以适当的方式执行或者测试/但不能对其他代码库造成任何损害的容器

你永远不应该同时添加allow-scripts和allow-same-origin到你的sandbox属性中——在这种情况下，嵌入式内容可以绕过阻止站点执行脚本的同源安全策略，并使用 JavaScript 完全关闭沙盒。

配置CSP指令
CSP：内容安全策略–提供一组http标头

embed/object
用于嵌入多种类型的外部内容的通用嵌入工具–java小程序/flash/pdf等插件/视频/svg/图像

插件：一种对浏览器原生无法读取的内容提供访问权限的软件

object–需要data属性/type

在网页中添加矢量图形

SVG矢量图：有较小的文件尺寸，但高度可缩放

位图：像素网格定义
.bmp/.png/.jpg/.gif

矢量图：算法定义–一个文件包含图片和路径–电脑计算出在屏幕渲染时应该呈现的样子
【有着更小的体积，仅需存储少量的算法，而不是逐个存储每个像素的信息】

SVG是什么
描述矢量图形的XML语言
有不同的元素来定义要显示在图像中的形状

SVG–标记图片/而不是内容

矢量图形编辑器

优点
矢量图像中的文字仍然可以访问–有利于SEO
SVG可以很好的适应样式/脚本
缺点
svg容易变得复杂–文件大小会增加
svg比栅栏图像更难创建

将svg添加到页面

  1. img 缺点：无法用js操作图像
      必须在svg代码中包含内联css样式
  不能用css伪类来重设图像样式

响应式图片

自适应图片

分辨率切换问题–在小屏幕手机上显示网页，没必要嵌入大的图片–浪费带宽

如何实现

srcset
–定义了允许浏览器选择的图像集，以及每个图像的大小
【文件名 空格 图像固有宽度w】

sizes
–定义了一组媒体条件，并指明当媒体条件为真时，什么图片尺寸是最佳选择
【媒体条件(可视窗口宽度) 空格 媒体条件为真图像将填充的槽的宽度】

浏览器

  1. 查看设备宽度
  2. 检查sizes列表中哪个媒体条件是第一个为真
  3. 查看给与该媒体查询的槽大小
  4. 加载srcset列表中引用的最接近所选的槽大小的图像

meta name=”viewport” content=”width=device-width”: 这行代码会强制地让手机浏览器采用它们真实可视窗口的宽度来加载网页

希望每个人在屏幕上看到的图片的实际尺寸是相同的
srcset结合css【设置宽度】

美术设计
pictrue
–sourse–media【相当于sizes】–srcset

为什么不能使用css/js实现这一效果
当浏览器开始加载一个页面，它会在主解析器开始加载和解析页面的 CSS 和 JavaScript 之前先下载 (预加载) 任意的图片。
–减少页面加载时间

不能先加载img元素后，再用js加测可视窗口的宽度，大小不合适再动态加载小的照片替换。
–原始的图像已经被加载了， 又加载了小图像

总结

  1. 美术设计–不同布局提供不同裁剪的图片–pictrue

  2. 分辨率切换–为窄屏幕提供更小的图片–sizes/srcset

HTML表单

是什么
用户和web站点或应用程序之间交互的主要内容之一–允许用户将数据发送到web站点

form
所有的HTML表单都以一个form开始

action：提交表单时，应该把收集的数据送给哪个模块/URL去处理

method：定义了发送数据的http方法

label:for–允许用户单机标签以激活响应的小部件
label 标签与input>通过他们各自的for 属性和 id 属性正确相关联（label 的 for 属性和它对应的小部件的 id 属性），这样，屏幕阅读器会读出诸如“Name, edit text”之类的东西。

label作用
使控件的可点击区域变大

fieldset–legend屏幕阅读器

input–空元素
设置默认值要用value属性

textarea
默认值直接写在里面

button
–type:submit/reset/button

向web服务器发送表单数据

form元素将定义如何通过action 属性和 method属性来发送数据的位置和方式。

name：收集指定的数据块
【浏览器端：告诉浏览器给数据各自哪个名称
服务器端：允许服务器按名称处理每个数据块】
