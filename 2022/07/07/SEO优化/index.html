SEO优化

好处

  1. 提高网站在有关搜索引擎内的自然排名，为自己的网站获取免费流量
  2. 以用户为核心，为目标用户提供高质量的服务

网页三要素的基础优化

  1. title–30字以内
    不同页面title不同
  2. description
    150字以内
    描述内容要和页面内容相关
    对页面内容的高度概括，不要过分堆砌关键词
    不同页面的description不同
  3. keywords
    为产品名，专题名，告诉搜索引擎本页的重点，关键词

总结
书写顺序
title>description>keywords

搜索引擎的索引方式

页面重定向和刷新

content代表时间–多少秒后刷新

语义化内容标签的使用

  1. a
    内部链接—加title说明
    外部链接—rel：nofollow–告诉蜘蛛无须追踪

  2. img
    加alt

  3. table
    标题用caption

  4. 标题
    h1一个页面只可有一个
    首页多用于包含logo

    h2–模块标题

    h3–段落的小节标题

  5. 强调内容的重要性
    strong/em

  6. 视觉上凸显文本
    mark

  7. br
    只用于文本内容的换行

语义化结构标签的使用

  1. section
    由标题和内容组成
    【不推荐没有标题的内容使用】

  2. header/footer
    article/section/aside/nav都可以拥有自己的header/footer标签

搜索引擎的注意内容

  1. 重要内容尽量往前放
  2. 重要内容不要用JS输出
  3. 页面结构不宜过深，最好不超过三级，每级都有面包屑导航
  4. 做404页面
